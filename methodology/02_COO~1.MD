# Co-occurrence Network Construction

A keyword co-occurrence network is a weighted graph used to represent relationships between keywords in the bibliometric corpus.

## Network Definition

In each network:

- A node represents a normalized keyword.
- An edge represents the co-occurrence of two keywords.
- An edge weight represents the strength of the co-occurrence between two keywords.
- A node degree represents the number of connections associated with a keyword.

A keyword with a high degree is connected to many other keywords and may therefore occupy an important structural position in the network. A pair of keywords connected by a thick edge co-occurs frequently and may indicate a strong thematic association.

## Construction Workflow

The keyword co-occurrence networks were constructed with the Sci2 Tool. After preprocessing and stemming, Sci2 creates a weighted network in which keywords are connected according to their co-occurrence patterns in the bibliometric corpus.

The Network Analysis Toolkit (NAT) was used to inspect each network. NAT provides direct information about the number of nodes and edges in the graph. Additional edge-weight summaries can be obtained through Sci2 weighted-network utilities or exported network tables.

## Weighted Edges

Edge weights are central to the interpretation of the graphs. They indicate the relative strength of the co-occurrence between two keywords. In the visualizations, stronger co-occurrence relationships are represented by thicker and more opaque edges.

For example, a strong edge between `metric` and `learn` indicates that these terms frequently appear together in the corpus and jointly represent the topic of metric learning.

## Node Importance

Node degree is used as a structural indicator of keyword importance. A node with many edges is connected to many other terms, suggesting that it participates in multiple thematic relationships.

However, node importance in the final visualizations also depends on whether the node survives the pruning process. A keyword displayed in a period-specific graph is interpreted as significant for that time window because it remains in the graph after pruning.

## Relationship to Later Steps

The constructed weighted networks are the input for:

1. Graph pruning, which improves readability and standardizes graph density across time periods.
2. Community detection, which identifies Level 0 and Level 1 thematic groups.
3. Graph visualization, which encodes keyword frequency, community membership, and co-occurrence strength.
